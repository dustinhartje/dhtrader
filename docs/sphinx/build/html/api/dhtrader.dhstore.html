dhtrader dhstore¶

Storage abstraction layer wrapping MongoDB persistence (dhmongo).

This module provides high-level storage functions for candles, trades, backtests, indicators, and events. It abstracts the underlying MongoDB implementation (dhmongo.py) to allow future migration to different storage solutions without changing higher-level code.

Core datetime-calculation utilities (expected_candle_datetimes, next_candle_start) are imported from dhcommon, which provides the base utility layer with no storage dependencies.
