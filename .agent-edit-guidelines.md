# Agent Edit Guidelines

This file intentionally stays minimal to avoid duplicated rule definitions.

Use these files in order:

1. `AGENTS.md` for workflow and rule scope
2. `CODING_STANDARDS.md` for canonical prose and command reference
3. Machine-enforced config (`.flake8`, `.editorconfig`, `setup.cfg`,
   `githooks/pre-commit`, `validate-file-quality.sh`)

If any conflict appears, follow the precedence in `AGENTS.md`.
